# ToDo App

A simple and clean task manager web application built using Flask, HTML, and CSS.  
The app allows users to add daily tasks, view their schedule, and delete completed or unwanted tasks.

## Live Demo

https://to-do-app-git-main-17-ankitas-projects.vercel.app/

## Features

- Add new tasks
- View all saved tasks
- Delete tasks
- Clean and responsive UI
- Flask-based backend
- HTML and CSS frontend
- Deployed using Vercel

## Tech Stack

- Python
- Flask
- HTML5
- CSS3
- Vercel
